# Vibe Flappy Bird

A polished Flappy Bird clone built with Python and Pygame-ce, featuring multiple difficulty levels, collectible coins, and dynamic background elements.

![Gameplay Screenshot](file:///Users/jeffzo/projects/practice/AIcoding/VibeCodingWithPython/010_flappy-bird/assets/bird.png)

## Features
- **Difficulty Selection**: Choose from Easy, Medium, or Hard modes.
- **Collectible Coins**: Gain extra points by grabbing shiny gold coins.
- **Dynamic Background**: Procedurally generated bubbly clouds.
- **Full Audio**: Includes background music and interaction sound effects.
- **Bird Physics**: Smooth movement with rotation based on velocity.

## How to Run
1. Ensure dependencies are installed:
   ```bash
   ./venv/bin/python3 -m pip install -r 010_flappy-bird/requirements.txt
   ```
2. Run the game:
   ```bash
   ./venv/bin/python3 010_flappy-bird/main.py
   ```

## Controls
- **Menu (Start Screen)**:
  - `1`: Start in **EASY** mode.
  - `2`: Start in **MEDIUM** mode.
  - `3`: Start in **HARD** mode.
- **Gameplay**:
  - `SPACE`: Flap wings to jump.
- **Game Over**:
  - `SPACE`: Return to difficulty selection menu.
